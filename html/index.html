How to use

  1. Download and install Bitcoin Core
  2. Download the latest snapshot of a pruned Bitcoin blockchain
  3. Extract it to your bitcoin folder
  4. Start Bitcoin Core

Notes:

Default Bitcoin data folder:

  • Linux: ~/.bitcoin
  • MacOS: ~/Library/Application Support/Bitcoin/
  • Windows: %APPDATA%/Bitcoin

The node is configured with prune=550 flag.
